## pizza login

Log into the CLI via GitHub

### Synopsis

Log into the OpenSauced CLI.

This command initiates the GitHub auth flow to log you into the OpenSauced CLI
by launching your browser and logging in with GitHub.

```
pizza login [flags]
```
